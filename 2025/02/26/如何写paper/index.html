如何写paper

发表于2025-02-26|更新于2025-03-25
|阅读量:

Notation

1、下标是index用 斜体
2、下标是专有名词用 \mathrm
3、括号用\left(\right)
4、sometimes you define notations in the main text. since you have this subsections, should define all notations here. Check. In main text, do not need to define again.

Abbreviation

缩写格式 example： estimating signal parameters via rotational invariance techniques (ESPRIT)
不要写成 Estimating Signal Parameters via Rotational Invariance Techniques (ESPRIT)
标题所有都要大写除了连词 example: \section{Cram\’ er-rao Bound}

破折号连接的形容词或名字

全文要统一，一般用小写
example： near-field,large-aperture
专有名字除外
example： mixed-ADC,root-MUSIC,Cramér-Rao bounds(CRB)

伪代码

伪代码的title只有首字母大写
example: Proposed near-field localization method

Remark

remark前的括号里只有首字母大写 (Computational complexity)
remark顶格写

Proof

最后加 thus completing the proof.

常用词汇短语

1、symmetricial->symmetric
2、rewirtten->re-expressed
3、target’s angles->the angles of targets 少用’s
4、(as will be shown in Section I) S大写
5、in the sequel
6、identify->differentiate
7、lead to
8、is mainly determined by
9、Next,
10、is attributed by
11、For ease of exposition
12、To this end为此

语言

1、要写具体，比如说算法性能、感知系统这种词太粗泛，尽量写成分辨率精度、定位系统这种
2、Benchmark里，介绍别人的工作，不要写成we can…. ，用第三人称去写
3、名词前有其他定语不要加the，example: the correct common spectrum peaks
4、少用let这种句子
5、冒号后大写，比如Proof: For
6、do not state “better performance”. this is not clear. what performance? rate? estimation?
7、do not say “similar performance’. Be more precise what performance.
8、if not using it in abtract, no need to define abbraviation.
9、when you express some catogrories, you can use DAs instead of the DA
10、少用we kown这种非正式表达
11、do not use “better”, if you can make it more concrete. e.g., larger? smaller? higher?
12、how to determine the values? all values in the paper should be justified.
13、少用can这种

引用

查询缩写：https://pubmed.ncbi.nlm.nih.gov/

会议论文：
@inproceedings{He03DCF,
author = {J. He and L. Zheng and Z. Yang and C. T. Chou},
title = {Performance analysis and service differentiation in {IEEE} 802.11 {WLAN}},
booktitle = {Proc. IEEE Conference on Local Computer Networks (LCN)},
month = {Oct.},
year = {2003},
address = {Bonn, Germany},
pages = {691—697},
}

期刊论文：
@article{Robinson04EDCF,
author = {J. W. Robinson and T. S. Randhawa},
title = {Saturation throughput analysis of {IEEE} 802.11e enhanced distributed coordination function},
journal = {IEEE J. Sel. Areas Commun.},
volume = {22},
number = {5},
pages = {917—928},
month = {June},
year = {2004},
}

Arxiv 文章：
@article{YouMixNearField,
author = {Yunpu Zhang and Changsheng You and Weijie Yuan and Fan Liu and Rui Zhang},
journal = {arXiv preprint arXiv:2304.07945},

year = {2023}}
